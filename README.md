# stm32_laser_car

这是我们在学习stm32时开发的一个项目：一个基于stm32的红外避障小车程序

## 原理
我们的思路是使用pwm调速控制马达进行调节。由于~~我们比较懒~~为了方便调试，我们没有将左右转的代码封装成函数。

>[!important]
>在``main.c``中你会发现在直线行驶时两个轮子的速度不一致，这是因为我们的小车配重原因自然右偏。

## 仓库中文件夹介绍
``Hardware``：存放各种外设库，虽然本项目我们只使用了pwm和motor两个库
``library和start`` ：存放stm32相关配置文件
``system``存放内设文件，虽然没有本项目也没有使用
``killkell.bat`` ：一个批处理文件，用于删除keil5编译时相关冗杂文件
更多内容请看根目录下的``代码报告.pdf``。同时，我们把做小车时所用到的资料一并上传到release中。
